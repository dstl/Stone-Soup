Liverpool Stone Soup Walk Tutorial
==================================

This is a step-by-step tutorial/guide to setting up Stone Soup for development purposes.

+---------------------------------------------+
| Target Software versions                    |
+==============+==============================+
| OS           | Windows 7, 8, 10 (MWS)       |
+--------------+------------------------------+
| Python       | 3.5, 3.6, 3.7                |
+--------------+------------------------------+
| Stone Soup   | any                          |
+--------------+------------------------------+

.. note::
    This installation tutorial is aimed for Windows 10 (both Generic and MWS) systems. However, the described process should be mostly identical for machines running on Windows 7 or 8.x.
